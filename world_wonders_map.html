Wonders of the Ancient & Modern World

A Leaflet exploration of humanity's greatest monuments

author: Data Explorer date: output: html_document runtime: shiny / leaflet

This document was produced using R Markdown and the leaflet package for R. It showcases an interactive world map pinpointing selected UNESCO World Heritage Sites and Wonders of the Ancient and Modern World — monuments that have captivated travellers and scholars for millennia.

Click any marker to learn more about the site. Use the layer control in the top-right corner to toggle between Ancient Wonders and Modern Wonders.

⚙ R Code

```{r setup, include=FALSE}
library(leaflet)
library(dplyr)

# Define wonder sites
wonders <- data.frame(
  name = c("Great Pyramid of Giza", "Colosseum",
           "Great Wall of China", "Machu Picchu", ...),
  lat = c(29.979, 41.890, 40.432, -13.163, ...),
  lng = c(31.134, 12.492, 116.570, -72.545, ...),
  type = c("Ancient", "Modern", "Modern", "Modern", ...)
)

# Build the map
leaflet(wonders) |>
  addProviderTiles("Stamen.Watercolor") |>
  addCircleMarkers(~lng, ~lat, popup = ~name, ...) ```
